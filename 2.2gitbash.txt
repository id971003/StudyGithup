#리눅스 #유닉스 #WINDOW에서리눅스커맨드이용가능

환경설정
git config user.name "name"
git config user.email "email"


로컬리포지토리생성
git clone "HttpPath"

add
git add "FilePath"

커밋
git committ -m "커밋메시지"

푸쉬
git push

풀
git pull

상태보기
git status

add 시킨 거 다시 빼기
git reset

다시 로컬리포지토리로 돌리기
git checkout -- "filename"

커밋메시지 수정
git commit --amend

유닉스 빠져나오기
esc > :qw!

강제 푸쉬
git push -f

이전 커밋 돌아가기 이전 커밋 정보 다지우면서 
git reset -hard "헤시값"

로그보기
git log

브랜치 status 종류 이름 가져옴
git bransh

브랜치만들기
git branch "name"

브랜치이동
git checkout "name"

브랜치 없애기
git branch -d "name"

브랜치 합치기
합칠 곳으로 가서  git marge "name"
